# facteur

A lightweight tool for managing CHANGELOG.md.

## Install

```bash
>$ npm install facteur --save-dev
#or
>$ yarn add facteur -D
#or
>$ pnpm add facteur -D
```

## Usage

### lint

```bash
>$ facteur lint <message>
```

### create

```bash
>$ facteur create [options]
```
